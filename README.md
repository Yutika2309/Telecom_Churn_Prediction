# Telecom Churn Prediction
Predicting whether customers will churn or not using classification algorithms such as Logistic Regression, SVM, Decision Tree, Random Forests and kNN.

## Libraries used:
* Pandas -> 1.0.5
* Numpy -> 1.18.5
* Matplotlib -> 3.2.2
* Seaborn ->  0.10.1
* Scikit learn -> 0.23.2
